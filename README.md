# Module5Tasks1_2

Отдельный Android Studio проект для практических заданий 1-2.

Что реализовано:

- задание 1: дневник, запись каждой заметки в отдельный файл во внутреннем хранилище приложения;
- список заметок загружается один раз при старте, добавление и удаление обновляют текущий список без полного пересканирования;
- задание 2: личная фотогалерея, съёмка фото через камеру, сохранение в `getExternalFilesDir(Environment.DIRECTORY_PICTURES)`;
- сетка фото в 3 колонки;
- экспорт фото в общую галерею через `MediaStore`.

Как запустить:

1. Открыть в Android Studio папку `Module5Tasks1_2`.
2. Дождаться Gradle Sync.
3. Запустить конфигурацию `app` на эмуляторе или телефоне.
